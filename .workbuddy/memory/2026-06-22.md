## v3.21.1: 修复全屏返回 + 快进快退 + 控制栏自动隐藏

### 问题
1. **全屏返回/侧滑返回直接退出 App**：Android `onBackPressed()` 使用异步 `evaluateJavascript` 检查 `isCSSFullscreen`，存在竞态 — JS 返回 null 或失败时 fallback 到 exit 逻辑，导致直接退出
2. **快进快退按钮不可见**：`.player-fullscreen` 的 `overflow: hidden` 裁剪了 Plyr 控制栏中的自定义按钮
3. **控制栏需要自动隐藏**：用户要求播放 3 秒后自动隐藏控制元素，暂停时显示

### 修复

#### 1. MainActivity.java `onBackPressed()` 同步检测
- 在全屏时直接检查 Java 层 `isFullscreen` 标志，同步调用 `exitFullscreenInternal()`
- 避免 `evaluateJavascript` 异步竞态导致按返回直接退出 App

#### 2. style.css overflow 修复
- `.player-fullscreen` 的 `overflow: hidden` → `overflow: visible`
- 确保 Plyr 控制栏（含快进快退按钮）在全屏时不裁剪

#### 3. 控制栏 3 秒自动隐藏
- CSS: `.plyr__controls` 添加 `opacity/visibility` 过渡动画，`.controls-hidden` 类控制隐藏
- JS `_initPlyrPlayer()`:
  - `player.on('play')` → 3 秒后添加 `controls-hidden` 类
  - `player.on('pause')` → 立即移除 `controls-hidden` 类
  - `mousemove`/`touchstart` → 显示控制栏并重置 3 秒计时
  - `cleanupPlayer()` → 清除 `_hideControlsTimer`

#### 4. 文件同步
- app.js / style.css 同步到 Android assets 目录
- 版本号更新为 v3.21.1
