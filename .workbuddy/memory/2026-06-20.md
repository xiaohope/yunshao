## 全屏方向自动适配

### 问题
用户反馈：点击全屏后视频没有横屏，还是在竖屏里播放；希望根据视频比例自动旋转——
横屏视频（电影/电视剧）强制横屏，竖屏视频（短剧）保持竖屏。

### 修改内容

#### 1. MainActivity.java — 新增 `setOrientation` JS 接口
在 `YunShaoNative` 对象里新增方法：
```java
@JavascriptInterface
public void setOrientation(String mode) {
    runOnUiThread(() -> {
        switch (mode) {
            case "landscape": setRequestedOrientation(ActivityInfo.SCREEN_ORIENTATION_LANDSCAPE); break;
            case "portrait":  setRequestedOrientation(ActivityInfo.SCREEN_ORIENTATION_PORTRAIT); break;
            default:           setRequestedOrientation(ActivityInfo.SCREEN_ORIENTATION_UNSPECIFIED); break;
        }
    });
}
```

#### 2. app.js — `enterFullscreenMode()` 增加方向检测
进入全屏前检测 `video.videoWidth / video.videoHeight`：
- `videoHeight > videoWidth` → 竖屏视频 → 调用 `YunShaoNative.setOrientation("portrait")`
- `videoWidth > videoHeight` → 横屏视频 → 调用 `YunShaoNative.setOrientation("landscape")`
- 元数据时未加载则监听 `loadedmetadata` 事件；500ms 超时默认横屏

#### 3. app.js — `exitFullscreenMode()` 恢复自动旋转
退出全屏时调用 `YunShaoNative.setOrientation("unset")` 恢复自由旋转。
